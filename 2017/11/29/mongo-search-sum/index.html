2017/11/29 mongo

Mongodb search aggregate SUM

  • find aggregate and where

db.cabinet.aggregate(
   [
     {
       $group:
         {
           _id: { },
           totalAmount: { $sum: "$cabinetBoxTotal" },
           count: { $sum: 1 }
         }
     }
   ]
)

   [
       { $match : { cabinetName : "中国农业大学西校区2号柜"} },
